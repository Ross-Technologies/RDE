# RDE

The official Ross Desktop Environment.

# Install

Required Packages:
* compton
* twmnd
* guake
* conky
* hsetroot
* gdesklets
* paramano
* nm-applet
* fbpanel
* pnmixer
* openbox

After installing the required packages, run `install.sh` as root. To start RDE run, `start-rde` in your xinit script.
